# Todo APIs (FastAPI + MongoDB)
A simple system to perform CRUD Operations using FastAPI and MongoDB

## Features

- Create one task
- Get list of all tasks
- Get single task (by ID)
- Update task status
- Delete task

---

## Tech Stack

- **Backend**: FastAPI
- **Database**: MongoDB (Local)
- **Language**: Python
- **API Docs**: Swagger (OpenAPI)

---
## How to run
- Start Mongodb - Connect
- pip install requirements.txt
- run 'uvicorn app.main:app --reload'

## API Testing
Testing can be done using the usual url: http://127.0.0.1:8000/docs
